**Subject:** `Re: [original thread subject]` — keep it in the same thread so it feels like a natural continuation of the conversation.

## Operational checklist (do before sending)

- [ ] Replace `[creatorname]` in the affiliate link with their actual handle (lowercase, no spaces — match the format `whop.com/meridian/?a=handle`)
- [ ] Replace `[CREATORNAME10]` with their promo code (typically uppercase channel name or handle + "10")
- [ ] Replace `[FirstName]` in the kit URL with their actual first name (for the personalized greeting on the kit page)
- [ ] Set up the Whop affiliate link before sending — don't send the email and then set it up later; the link must be live when they click it
- [ ] Activate their Guard license in Whop (Comp a subscription → their email) — within 24h of sending
- [ ] Log the creator as `status: accepted` in your outreach tracking

---

## Whop operations reference (do these before or immediately after sending)

**To generate the affiliate link:**
1. Go to your Whop seller dashboard
2. Affiliates → Add affiliate
3. Enter the creator's email and set the affiliate handle (the `?a=` parameter)
4. Whop generates the link automatically

**To generate the promo code:**
1. Whop seller dashboard → Promotions → Create discount
2. Set to 10% off, recurring (applies to every renewal), no expiry
3. Name the code `[CREATORNAME10]` or similar
4. Assign it to the affiliate

**To comp the Guard license:**
1. Whop seller dashboard → Members → Add member
2. Enter the creator's email
3. Select Meridian Guard → Complimentary (no charge)
4. Set duration: "Active partnership" — review quarterly

## Tone notes

- Keep the email short and operational — they said yes, they don't need more selling
- The kit URL with `?name=` personalizes their greeting on the kit page — use it
- "No pressure on timeline" is genuine — creators who feel rushed to post often produce lower quality content; let them get familiar with the product first
- Don't add anything to this email that isn't in the template above without good reason — every extra sentence is friction
