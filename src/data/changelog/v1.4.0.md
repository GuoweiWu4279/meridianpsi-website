Minor release. Recommended for all users on v1.3.3 or earlier. No profile, `guard.xml`, or session data migration required.

### What's new

**Guard — real trading pause (enforcement)**

- **Trading Pause** — block new entries for a set duration or rest of day.
  - **Cancel orders** — stay connected; new entry orders are cancelled automatically.
  - **Disconnect** — sever the broker connection; optional **re-disconnect** if you reconnect during the pause.
- **Auto-flatten on pause** — optional close of all open positions when pause starts (Cancel and Disconnect modes).
- **Acknowledge** — optional countdown before the OK button unlocks; optional phrase to type.
- **Risk Alert** — persistent HUD warning while the condition is active.
- **Notify** — toast with configurable duration.
- **Min loss filter** — consecutive-loss rules can ignore small losses (fee-sized scratches).
- **Guard UI** — action-first rule editor, clearer labels, pause mode cards, active pause countdown in Dashboard.

Legacy **Disconnect** rules in existing `guard.xml` files still work and map to Trading Pause (Disconnect mode).

### Fixes

**Guard — pause and reconnect**

- Trading Pause / Disconnect now enforce as designed (confirmation default, auto-flatten in Cancel mode, acknowledge countdown).
- **PSI no longer jumps** when you reconnect during a Disconnect-mode pause.
- **Pause survives NT8 restart or crash** — if pause time has not expired, protection resumes on next startup.

**Guard — conditions (also in v1.3.3)**

- Consecutive-loss condition fires when NinjaTrader sends the close fill before the position REMOVE event.
- Emergency Override no longer makes the Guard window pop back immediately.

### Who should upgrade

- **Anyone using Guard** for discipline or risk limits — this is the largest Guard update since launch.
- **v1.3.3 users** — direct upgrade; keep your existing rules.

### How to upgrade

1. Tools > Import > NinjaScript Add-On > select `MeridianPSI_V1.4.0.zip`.
2. Restart NinjaTrader if prompted.
3. Control Center > New > Add-on > Meridian Dashboard.
4. Review **My Rules** in the Guard tab — new Trading Pause options are available when editing a rule.
