**Minor release.** Get protected from your very first session, plus fixes to several Dashboard panels that showed stale or confusing values. Recommended for all users.

### New

- **First-run Quick Setup** — on first launch, Meridian now walks you through your three protective limits (daily loss limit, pause-after-N-losses, max position size) and your session window. This means the strong **Class A** signals — the ones that move PSI 30–40 points — are active from session one, instead of staying silent until you find them in Settings. Takes about 30 seconds; everything stays editable in Settings, and the wizard only appears when no profile has been set up yet.

### Fixes

- **Profile — "Last Session PSI" now matches** the Session report and History. It previously showed a stale value that disagreed with the rest of the app.
- **Profile — calibration text now tells you what's actually left** (e.g. *"30% calibrated — needs 21 more trades"*) instead of the confusing *"0 more sessions."*
- **Guard — clear status when on but unprotected.** If Guard is enabled but no rule is active, the page now says *"Guard is on, but no rules are active — enable one below to be protected,"* so a "Guard ON" toggle never gives a false sense of safety.
- **Session — "Meridian is watching" card** on the pre-trade empty state, showing the limits currently protecting you (daily loss, loss-streak pause, max size, trade count, session window) with a link to adjust them.

### Who should upgrade

- **All users.** New users get a protected setup out of the box; existing users get the accuracy and clarity fixes. Users who already configured their limits will not see the wizard.

### How to upgrade

1. Tools > Import > NinjaScript Add-On > select `MeridianPSI_V1.4.2.zip`.
2. Restart NinjaTrader if prompted.
3. Your existing profile, Guard rules, and history are kept — no migration needed.
