### What's New

- Initial public release of Meridian for NinjaTrader 8
- Real-time psychological stability index panel (0–100 composite score)
- Session tracking with automatic session detection on market open/close
- Configurable alert thresholds with in-platform visual and audio notifications
- Session history log with basic statistics (average score, lowest score, session duration)

### System Requirements

- NinjaTrader 8.0.23.0 or later
- Windows 10 / 11 (64-bit)
- .NET Framework 4.8

### Known Issues

- Session auto-detection may require manual reset after NinjaTrader restart on some configurations
